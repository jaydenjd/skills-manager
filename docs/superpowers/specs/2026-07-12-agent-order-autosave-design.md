# Agent 排序与设置自动保存设计

## 目标

- 左侧 Agents 区域固定将“全部 Agents”显示在第一位。
- 其他 Agent 严格按照设置中 `sources` 的配置顺序显示，不再根据已安装 skill 数量或名称排序。
- 设置页中的 Agent 支持上下拖动排序。
- 移除“保存设置”按钮，所有设置改动自动生效并持久化。

## 范围

本次调整覆盖设置页的可视化模式与 JSON 模式、Agent 排序交互、左侧 Agent 导航排序，以及自动保存状态和错误反馈。不改变 skill 扫描、安装目标选择、版本管理或来源数据结构。

## 数据模型与排序

`settings.sources` 数组是 Agent 顺序的唯一来源。拖拽只改变数组内元素的位置，不修改 source 的 `id`、名称、目录或启用状态。

左侧导航按以下规则生成：

1. “全部 Agents”始终由界面单独渲染并固定在第一位。
2. 遍历 `settings.sources` 中启用的 source，按数组顺序渲染对应 Agent。
3. 每项数量从扫描结果按 source 的 client 统计；没有已安装 skill 的已启用 Agent仍显示，数量为 0。
4. 扫描结果中存在、但当前配置中不存在的来源作为兼容性兜底追加到末尾，避免数据暂时不同步时入口消失。

## Agent 拖拽交互

设置页每个 Agent 行增加明确的拖拽手柄。使用浏览器原生拖放事件实现，避免为单一列表引入新的拖拽依赖。

- 只有从手柄开始操作才进入拖拽排序，避免编辑输入框时误触。
- 拖动经过其他行时展示目标位置提示。
- 放下后更新 `draft.sources`，并立即触发设置保存。
- 拖拽未改变位置时不写配置。
- 键盘与无拖拽环境提供“上移/下移”按钮，执行相同的重排和保存逻辑。

## 自动保存策略

设置页保留本地 draft 以保证输入流畅，并通过统一自动保存通道写入后端。

- 开关、下拉框、新增、删除、拖拽和上移/下移：状态变更后立即保存。
- 名称、描述、目录、忽略规则及数字输入：停止输入 500ms 后保存。
- 多次快速修改合并为最后一次保存；若保存正在进行，新变更在当前请求结束后继续保存，避免旧响应覆盖新状态。
- 成功保存后使用服务端规范化结果更新页面顶层 settings，但不显示高频成功通知。
- 保存失败时保留 draft，显示现有失败通知；后续修改可再次触发保存。
- 页面首次载入 settings 或服务端保存结果回填时不重复触发保存。

## JSON 模式

JSON 编辑器继续以文本为本地状态：

- 输入后等待 500ms，再尝试解析。
- 仅当内容是合法 JSON 对象时自动保存。
- 解析失败时显示行内错误，不调用保存接口，也不覆盖当前有效设置。
- 从 JSON 模式切换到可视化模式时，若 JSON 无效则阻止切换并保留错误；有效时先应用并保存解析结果。
- 从可视化模式切换到 JSON 模式时，将当前 draft 序列化到编辑器。

## 组件边界

- `SettingsPage`：维护 draft、编辑模式、拖拽状态与自动保存调度。
- 顶层 `App`：继续负责调用设置 IPC、更新全局 settings、刷新扫描结果和展示失败通知；保存函数返回服务端保存后的 settings。
- 左侧导航派生数据：结合 `settings.sources` 与扫描结果计算有序的 Agent 计数列表。
- Electron 设置存储：继续按收到的 `sources` 数组顺序保存，无需新增持久化字段。

## 错误与并发处理

- 使用递增修订号或等效的串行队列识别最新 draft，确保较慢的旧保存不会覆盖较新的编辑。
- 自动保存失败不回滚输入内容。
- 删除当前默认安装来源时，沿用现有逻辑选择第一个启用来源；这一变更与删除操作一同保存。
- JSON 校验错误与后端保存错误分开呈现，便于用户判断是格式问题还是持久化失败。

## 测试与验收

- 左侧“全部 Agents”始终第一，其余项按 `settings.sources` 顺序展示。
- 修改配置顺序后，左侧顺序随保存结果更新，重启应用后保持一致。
- 拖拽及上移/下移均能改变顺序，并且不改变 source 内容。
- 开关和下拉变更立即保存；连续文本输入只在防抖结束后保存一次最终值。
- 保存进行中继续编辑时，最终持久化的是最新值。
- JSON 无效时不保存；恢复有效后自动保存。
- 设置页不再显示“保存设置”按钮。
- 执行项目现有测试（若有）及 `npm run build`。

## 非目标

- 不支持将“全部 Agents”参与拖拽。
- 不增加多套自定义排序方案。
- 不改变禁用 Agent 的扫描和安装语义。
- 不重构与设置保存无关的页面或 Electron API。
